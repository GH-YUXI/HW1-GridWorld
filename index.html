n x n 網格地圖與 Value Iteration

先設定起點、終點與障礙物，再執行 Value Iteration，直接在頁面上顯示每個狀態的最佳策略與狀態價值 V(s)。

編輯模式：
目前 n：5
起點數：0/1
終點數：0/1
障礙物：0/3
Reward：-1
γ：0.9
目前模式：起點
收斂門檻：1e-4
起點
終點
障礙物
一般格子

操作說明

  1. 選擇 n（5 到 9）並建立網格。
  2. 設定步驟獎勵 Reward 與折扣因子 γ。
  3. 切換模式後，點擊格子設定起點、終點、障礙物。
  4. 障礙物數量最多為 n-2，可少於 n-2。
  5. 按下按鈕後，格子會顯示最佳策略箭頭與 V(s)。

請先建立網格並完成設定。
