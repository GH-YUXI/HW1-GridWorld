# Value Iteration 說明

## 核心概念

Value Iteration 是一種用來求解 Markov Decision Process (MDP) 最佳策略的方法。
它會反覆更新每個狀態的價值函數 `V(s)`，直到收斂，再依據價值函數挑出最佳動作。

## 更新公式

```math
V_{k+1}(s)=\max_a \left[r(s,a,s') + \gamma V_k(s')\right]
```

其中：
- `s`：目前狀態
- `a`：可採取動作
- `s'`：下一狀態
- `r`：即時回饋
- `γ`：折扣因子

## 本專案中的環境設定

- 動作集合：`上、下、左、右`
- 撞牆或撞障礙物：停在原地
- 抵達終點：reward = 0
- 其他移動：reward = stepReward
- 終點價值固定：`V(goal)=0`

## 同分時隨機選擇最佳動作

在某些狀態中，可能有多個動作都得到相同的最佳值，例如：

```text
Up   -> -3.2
Right-> -3.2
Down -> -4.1
Left -> -5.0
```

此時最佳動作集合為：

```text
[Up, Right]
```

本專案會從這個集合中隨機選一個顯示在畫面上，因此每次執行時，箭頭可能不同，但都仍屬於最佳策略。
